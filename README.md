# toast-notification
A Toast Notification with modes. Coded with Vanilla JS

to use:

```
createNotification(title, body, type)
```
An Example:
```
createNotification("Success", "This is a success notification", "success")
```

## Modes
type can be:
- success ✅
- error ⛔
- warning ⚠
- info ❕
